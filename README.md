# Hello, User! (ANSI C)

Asks the user's name and displays a greeting message it or the default name `"World"`.

```sh
$ ./main
Type your name:
Alisson
Hello, Alisson!
```

or 

```sh
$ ./main
Type your name:

Hello, World!
```
